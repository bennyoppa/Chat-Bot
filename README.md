# Chat-Bot
COMP9900 project
Team: Humble Learners

How to run the project:
1. unzip mychatbot.zip
2. install all required python libraries (requirements.txt)
3. make sure the machine environment supports django project
4. under the directory of manage.py, type 'python manage.py runserver'
5. open the link 'http://127.0.0.1:8000/UI' in a web browser
