# Chat-Bot
COMP9900 project
Team: Humble Learners

How to run the project:
1. unzip mychatbot.zip
2. install all required python libraries (requirements.txt)
3. install language pack for spacy by typing 'python -m spacy download en'
4. make sure the machine environment supports django project
5. under the directory of manage.py, type 'python manage.py runserver'
6. open the link 'http://127.0.0.1:8000/UI' in a web browser
